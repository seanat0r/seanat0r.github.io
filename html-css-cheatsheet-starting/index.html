Head, DOCTYPE and html

Definition: Use Case: Example:
<!DOCTYPE html> It tag say, that this is Document is a html version 5 document <!DOCTYPE html> (It doesen't need an closing tag)
<html> Represents the entire HTML document <html> <!-- Entire document goes here --> </html>
<head> This Tag is for evrything, that you can't see on the website. It have one exepticon, that is for the title tag <head> <!-- Metadata comes here ---> </head>
<link> Use it for connecting other external file such as stylesheet or a library. The 2 main attributs are rel="" and href="" <link rel="stylesheet" href="some/outside/source.css" />
<title> Use this for the tab-title. On this page the title is: HTML and CSS Cheat Sheet <title>HTML and CSS Cheat Sheet</title>
<meta> Defines metadata that doesn't appear in the visible content of the webpage but is used by browsers, search engines, and other web services. <meta charset="UTF-8">
<base> Defines the base URL for all relative URLs in an HTML document. <base href="https://example.com/" target="_blank">
<script> Used to embed or reference JavaScript code within the document. <script src="script.js"></script>
<style> Defines CSS styles directly within the HTML document. <style> body { background-color: lightblue; } </style>
<noscript> Defines a section in the document that is displayed when JavaScript is disabled in the browser. <noscript> JavaScript is disabled in your browser. </noscript>

CSS

Basic

To write a CSS code, you need to follow the following syntax:
p {
color: green;
}
As you see the first letter (p) defines what you want to change. The second is what you want to change, in this example we change the color from the letter from white (standard) to green.

Color

Definition: Use Case: Example:

background-color Change the background of the object p { background-color: blue; }
rgb(red, blue, green) & rgba(r, b, g, alpha) defines the value of the color. The first three number can be from 0 to 255. The fourth number can be from 0 to 1, it change it opacity p { color: rgb(100, 200, 300); }
